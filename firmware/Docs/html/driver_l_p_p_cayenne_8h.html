Detailed Description

Encodeur de payload au format Cayenne LPP pour LoRaWAN.

Author
[Ton Nom/Projet]

Ce module permet de construire facilement une trame binaire compatible avec la norme Cayenne Low Power Payload. Il intègre des sécurités contre le dépassement de buffer et l'overflow mathématique des capteurs.
