Detailed Description

Pilote asynchrone pour module LoRaWAN commandé par AT.

Author
[Ton Nom/Projet]

Ce driver implémente une machine d'état (FSM) 100% non-bloquante basée sur l'UART DMA. Il gère la configuration, la jonction au réseau (OTAA), l'envoi de payloads (Hexadécimal), ainsi que la gestion stricte de l'énergie (Sleep / WakeUp).
