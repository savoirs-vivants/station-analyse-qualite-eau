\doxysubsection{Detailed Description}
Pilote d\textquotesingle{}acquisition analogique (ADC) asynchrone avec monitoring. 

\begin{DoxyAuthor}{Author}
\+[Ton Nom/\+\+Projet]\+ 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
\+[Date actuelle]\+
\end{DoxyDate}
Ce module gère l\textquotesingle{}acquisition de capteurs analogiques externes et internes (Température MCU, VREFINT, VBAT) via le DMA du STM32 pour un fonctionnement 100\% non-\/bloquant.
\begin{DoxyItemize}
\item Il intègre une compensation dynamique de la tension d\textquotesingle{}alimentation (VDDA) basée sur la lecture de la référence interne (VREFINT), garantissant des mesures précises même lors de la décharge d\textquotesingle{}une batterie.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
L\textquotesingle{}ordre des conversions dans Cube\+MX doit correspondre aux index :\+
\begin{DoxyItemize}
\item Index 0, 1 :\+ Capteurs externes (ex:\+ C1, C2)
\item Index 2 :\+ VREFINT Channel
\item Index 3 :\+ Temperature Sensor Channel
\item Index 4 :\+ Vbat Channel 
\end{DoxyItemize}
\end{DoxyNote}
