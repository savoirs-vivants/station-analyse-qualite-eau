\doxysubsection{Detailed Description}
Pilote asynchrone pour module Lo\+Ra\+WAN commandé par AT. 

\begin{DoxyAuthor}{Author}
\+[Ton Nom/\+\+Projet]\+
\end{DoxyAuthor}
Ce driver implémente une machine d\textquotesingle{}état (FSM) 100\% non-\/bloquante basée sur l\textquotesingle{}UART DMA. Il gère la configuration, la jonction au réseau (OTAA), l\textquotesingle{}envoi de payloads (Hexadécimal), ainsi que la gestion stricte de l\textquotesingle{}énergie (Sleep /\+ Wake\+Up).
